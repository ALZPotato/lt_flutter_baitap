
### 1. Cấu hình Backend (Node.js+Atlas MongoDB)
*   **Lưu ý bảo mật:** File `.env` đã được ẩn. Quý thầy/cô vui lòng tạo file `.env` mới và điền biến `MONGO_URI` (kết nối MongoDB) và `JWT_SECRET` để server hoạt động.

### 2. Cấu hình Frontend (Flutter)
*   Kiểm tra IP trong `lib/services/api_service.dart` để khớp với IP server.
